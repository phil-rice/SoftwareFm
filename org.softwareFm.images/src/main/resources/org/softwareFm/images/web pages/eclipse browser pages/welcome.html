SoftwareFM is an open source Eclipse plug-in, for Java developers, that accesses a crowd sourced database of information relating to software projects

SoftwareFM listens to selection events within the Eclipse workspace and presenting context sensitive information, news and views, about the specific jar / artifact / project

The ethos at SoftwareFM is co-operation and helping each other, so please get actively involved by adding to the crowd sourced database of information, on the projects you use
