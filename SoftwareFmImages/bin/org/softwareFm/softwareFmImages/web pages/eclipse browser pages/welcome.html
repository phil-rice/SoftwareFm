Welcome to Software FM an Open Source Eclipse plug-in for Java Developers

SoftwareFM works by listening to selection events within the workspace, and presenting context sensitive informtation about the specific jar / artifact / project the thing you selected was defined in.

When you start selecting, this view will change:

  • Inforamtion will be shown on this view. Clicking in the view will tell you more.
  • The right hand section of this view will tell you news and information about the selected project.
  • As you and others add data, SoftwareFM will get better at feeding relevant data and news.
